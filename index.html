Develop Image Processing Algorithms

In the following algorithms we will be transmitting two pixels at once for faster processing speed. Before the images can be processed they have to be converted to bmp first then to hexadecimal as verilog cannot read images directly. The BMP format is often used in image processing and computer vision applications because it provides an uncompressed representation of an image’s pixel values, making it easy to extract and manipulate the pixel values. Additionally, BMP files have a simple file structure, which makes them easy to work with.

Brightness Addition/Subtraction

The brightness addition/subtraction algorithm works by adding or subtracting a constant value to every pixel in an image, which changes the brightness level of the entire image. When a positive constant is added to every pixel, the brightness level of the image is increased. This means that the image appears brighter, with lighter colors and more contrast between the different shades of colors. On the other hand, when a negative constant is subtracted from every pixel the brightness level of the image is decreased. This means that the image appears darker, with darker colors and less contrast between the different shades of colors.

Inverting

The inverting algorithm works by reversing the brightness values of the pixels in the image. This is done by subtracting each pixel’s brightness value from the maximum possible brightness value, here it is 255.

Thresholding(Black/White)

Thresholding is a simple technique that converts a grayscale image to a binary image, where the pixels are either black or white. This is done by setting a threshold to white and all the pixels with values below the threshold to black.
